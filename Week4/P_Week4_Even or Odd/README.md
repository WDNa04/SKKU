*Exercise

Write a code that takes an integer as input and outputs whether it is **even** or **odd** <ins> if it is positive</ins>, and outputs **the input value** as it is <ins> if it is 0 or negative</ins>.

*Example

1)
input
<pre>
2023
</pre>
output
<pre>
odd
</pre>
2)
input
<pre>
-21
</pre>
output
<pre>
-21
</pre>
3)
input
<pre>
22
</pre>
output
<pre>
even
</pre>
